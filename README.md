# Customer Shopping Behavior Analysis

## Overview

This project analyzes customer shopping behavior using Python, SQL Server, and Power BI. The goal is to identify purchasing patterns, customer segments, product performance, and business insights through data cleaning, exploratory data analysis (EDA), SQL querying, and dashboard visualization.

The project follows a complete data analytics workflow, from raw data processing to business reporting and presentation.

## Dataset

**Dataset Name:** Customer Shopping Behavior Dataset

The dataset contains customer purchase information, including:

* Customer demographics
* Product details
* Purchase amount
* Purchase frequency
* Review ratings
* Subscription status
* Discount information
* Previous purchase history
* Payment methods

## Project Workflow
### 1. Data Loading

* Imported dataset into Python using Pandas.
* Performed initial data inspection and validation.

### 2. Exploratory Data Analysis (EDA)

* Analyzed customer demographics.
* Examined purchase behavior and spending trends.
* Identified top-selling products and categories.
* Evaluated customer ratings and subscription patterns.

### 3. Data Cleaning & Feature Engineering

* Handled missing values.
* Standardized categorical data.
* Created derived columns such as:

  * Age Groups
  * Purchase Frequency (Days)
  * Customer Segments

### 4. SQL Analysis

Data was imported into SQL Server for advanced querying.

Key analyses included:

* Customer segmentation
* Top purchased products
* Discount impact analysis
* Category-wise product rankings
* Customer loyalty analysis
* Revenue and purchase trends

### 5. Power BI Dashboard

Built an interactive dashboard to visualize:

* Total Revenue
* Average Purchase Amount
* Customer Segments
* Product Category Performance
* Discount Analysis
* Purchase Frequency Trends
* Customer Demographics

### 6. Reporting & Presentation

* Created a detailed analytics report summarizing findings.
* Designed a professional presentation using Gamma AI.

## Dashboard Highlights

Key KPIs:

* Total Customers

Interactive filters allow users to explore insights by category, customer demographics, and purchasing behavior.

## Key Results

* Identified the most purchased products across categories.
* Segmented customers into New, Returning, and Loyal groups.

## How to Run

### Python Analysis

1. Clone the repository.
2. Install required libraries:

```bash
pip install pandas numpy matplotlib seaborn pyodbc sqlalchemy
```

3. Run the Jupyter Notebook:

```bash
jupyter notebook
```

### SQL Analysis

1. Open SQL Server Management Studio.
2. Create a database.
3. Import the cleaned dataset into SQL Server.
4. Execute SQL scripts for analysis.

### Power BI Dashboard

1. Open the Power BI (.pbix) file.
2. Connect to the SQL Server database.
3. Refresh the data model.
4. Explore the dashboard visuals and insights.
